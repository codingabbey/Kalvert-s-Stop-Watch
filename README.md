A simple, lightweight stopwatch application built with vanilla JavaScript. Perfect for basic time tracking tasks.

Features:
Start/Stop: Begin and pause the timer with dedicated buttons
Reset: Clear the timer back to 00:00
Time Display: Shows elapsed time in MM:SS format with zero-padding
Minimal Design: Clean, straightforward interface with no distractions

How It Works:
Open index.html in your web browser
Click Start to begin the timer
Click Stop to pause it
Click Reset to return to 00:00

Current Limitations:

No lap/split functionality
No milliseconds display (only MM:SS)
No persistent timer state (resets on page refresh)
No visual feedback or styling
No keyboard controls
Single timer instance only

Potential Additions:

Lap Times: Add a "Lap" button to record split times
Milliseconds: Extend display to MM:SS:MS for more precision
Keyboard Shortcuts:
Space to start/stop
R to reset
L to record lap
Styling: Add CSS for a more polished look
Local Storage: Save timer history across sessions
Audio Notification: Alert sound when timer completes
Multiple Timers: Support running multiple stopwatches
Time Validation: Prevent multiple start clicks
Display Toggle: Switch between elapsed time and countdown modes
